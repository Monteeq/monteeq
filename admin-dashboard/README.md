# Monteeq Admin Dashboard

This admin dashboard is built with React and Vite only. It does not use Next.js.

## Scripts

- `npm run dev` starts the local Vite development server
- `npm run build` creates a production build
- `npm run preview` serves the production build locally

## Development notes

- The app is configured in [vite.config.js](vite.config.js)
- The dev server runs on port 5174
- API requests are proxied to the backend at http://localhost:8000
